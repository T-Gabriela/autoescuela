Sistema de Gestión de Autoescuela

Requisitos

- **Servidor Web**: Apache 2.4 o superior
- **PHP**: Versión 7.4 o superior
- **MySQL**: Versión 5.7 o superior
- **WampServer**: 3.3 o superior (recomendado)
- **Navegador**: Chrome, Firefox, Edge o Safari actualizado
